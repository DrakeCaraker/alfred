# Project

This is a fresh project. Run `/bootstrap` to generate your personalized CLAUDE.md.

Bootstrap asks you 3 questions (your role, coding comfort, what you're
building), then replaces this file with a personalized project guide —
domain-specific guardrails, a directory map, tool recommendations, and
references for every command. Takes about 2 minutes.

## Quick Start

1. Run `/bootstrap` to set up your development environment
2. Run `/teach` to learn the first development habit
3. Run `/status` to check your progress

Once you run bootstrap, this file becomes the project's AI instruction set —
read by Claude at the start of every session.
